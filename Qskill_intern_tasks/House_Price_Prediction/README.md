# House Price Prediction

## Objective
Predict house prices using machine learning regression techniques.

## Dataset
California Housing Dataset from Scikit-Learn.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Steps Performed
1. Loaded and explored dataset
2. Checked missing values
3. Normalized features
4. Split data into training and testing sets
5. Trained Linear Regression model
6. Evaluated using MSE and R² Score
7. Visualized results

## Model Performance

Mean Squared Error (MSE): 0.556

R² Score: 0.576

The Linear Regression model explains approximately 57.6% of the variance in house prices.
